Readme.md

Sharp-Html (SH) is a C# library to help those of us who are writing quick and dirty reports, or other text based "stuff" in C# and need to display that "stuff" in HTML. Specifically if you use a lot of tables especially identical tables SH allows you to create table templates that you can use over and over, without having to keep redefining them.

To get started look at the Examples project.

There are the raw tags themselves.

There is a BasicPage class.

There is a SimplePage class.

this gives you a basic skin
